# Brand Aesthetic: [SIC]

## Rol del Documento
Este archivo define el sistema visual de [SIC]. Su función es traducir la identidad de marca a reglas concretas de diseño para web, piezas promocionales, portadas, visualizers, social media y materiales editoriales.

## Concepto Visual
**Alto Contraste Editorial**

La estética de [SIC] nace de la tensión absoluta entre luz y oscuridad. El rojo aparece como única señal vital dentro de un sistema casi binario. El resultado debe sentirse minimalista, hostil, preciso y ligeramente contaminado por una imperfección humana o grunge.

## Principios Visuales
- contraste duro, no suave
- composición con intención minimalista y abastracta
- blanco y negro como base dominante
- rojo como acento escaso y táctico
- sensación de documento, manifiesto o evidencia intervenida
- imperfección controlada, nunca desorden gratuito

## Paleta de Color
Solo se permite esta paleta base:

- **Negro Absoluto:** `#000000` o `#0A0A0A`
- **Blanco Crudo:** `#F5F5F5` o `#FFFFFF`
- **Rojo Visceral:** `#D21F3C`

## Reglas de Uso del Color
- el negro es el fondo predominante
- el blanco sostiene texto, estructura y respiración
- el rojo solo se usa para acción, corrección, énfasis, error o pulso visual
- no usar grises medios como color principal del sistema
- no introducir colores complementarios o decorativos
- blanco y negro se pueden invertir (fondo-texto)

## Tipografía
### Títulos Principales
**Syncopate**

Uso:
- titulares
- nombres de secciones
- frases de alto impacto

### Títulos Estructurales
**Audiowide**

Uso:
- labels técnicos
- bloques conceptuales
- elementos de interfaz con tono de laboratorio

### Cuerpo e Interfaces
**Orbitron**

Uso:
- párrafos cortos
- navegación
- UI
- fichas técnicas

### Tratamientos
- usar cursivas para insinuar ambigüedad, citas internas o voces alternas y usar tachados para simular "clasificado"
- evitar mezclas tipográficas adicionales sin una razón de sistema

## UI y Composición
- bordes de 1px, rectos
- sin esquinas redondeadas
- preferir retículas rígidas
- usar mucho espacio negativo
- priorizar jerarquías claras y cortes bruscos
- evitar sombras suaves y recursos "amables"

## Tratamiento de Imagen
- imágenes en escala de grises, bitonal o duotono negro/rojo
- alto contraste antes que fidelidad fotográfica
- textura de ruido o dithering cuando aporte tensión
- evitar fotografía cálida, saturada o naturalista

## Logo
- blanco sobre negro o negro sobre blanco
- nunca en rojo
- nunca con degradados
- nunca como elemento decorativo repetido sin función compositiva

## Texturas y Superficie
- ruido digital fino
- estática visual
- suciedad editorial controlada
- marcas de censura, bloques, tachones o redacciones cuando el concepto lo justifique

## Referencias
- constructivismo
- fanzine político
- documentos censurados
- cine noir de alto contraste
- interfaces de laboratorio o archivo intervenido

## Criterio de Validación
Una pieza visual de [SIC] está bien dirigida si:
- puede reconocerse a distancia por contraste y restricción
- no depende de ornamento genérico
- transmite tensión intelectual y emocional
- se siente más cerca de un manifiesto o una obra de arte abastracta que de una pieza promocional convencional
